# Library Management System

Blazor Server application for managing library borrowing and returning.

## Tech Stack

- .NET 8
- Blazor Server
- Entity Framework Core
- SQL Server

## Features

Student:
- Login
- Search books
- Borrow books

Admin:
- Manage books
- View borrow records

## Database

Tables:
- Users
- Books
- BorrowRecords
- Sessions
